Server

The server component provides a RESTful API for tracks. It is written in Java and published under GNU Affero General Public License, Version 3 or later.

Website

The website provides general information about the project as well as the user pages and links to analyses and maps. It is written in PHP and published under GNU General Public License, Version 3.

App

Android app for collecting data from smartphone sensors and OBD-II interface in a car. Tracks can be uploaded to the enviroCar server. The app is published under GNU General Public License, Version 3.
